- ✅ **卡片学习模式** - 200个四六级核心词汇，支持翻转查看释义

## 🚀 如何使用

### 1. 本地测试（当前可用）
```bash
cd /Users/arcade/Desktop/english-flashcards_副本

# 开发模式
npm run dev
# 访问: http://localhost:5173

# 生产预览
npm run build
npm run preview
# 访问: http://localhost:4173
```

### 2. 手机测试
1. 确保手机和电脑在同一 WiFi
2. 运行 `npm run dev`
3. 用手机浏览器访问: `http://你的电脑IP:5173`
4. 测试"添加到主屏幕"功能

### 3. 部署到线上（推荐 Vercel）
```bash
# 安装 Vercel CLI
npm i -g vercel

# 一键部署
vercel

# 或者直接在 vercel.com 导入 GitHub 仓库
```

1. 用 Chrome 打开应用

4. 点击"安装"

## 🎯 技术栈

- **框架**: React 19 + Vite 8
- **样式**: Tailwind CSS 4
- **PWA**: vite-plugin-pwa + Workbox
- **存储**: localStorage
- **语音**: Web Speech API
- **动画**: CSS 3D Transforms

## 🐛 已知限制

1. **TTS 语音**
   - 依赖系统 TTS 引擎
   - 不同设备音质可能不同
   - 需要联网下载语音包（首次）

2. **存储限制**
   - localStorage 限制 5-10MB
   - 当前使用量 < 1KB
   - 足够存储进度数据

3. **浏览器兼容性**
   - iOS 需要 Safari 11.3+
   - Android 需要 Chrome 40+
   - Service Worker 需要 HTTPS
